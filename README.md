# Habit-Guard
Habit-Guard is a mobile habit tracking app built with Expo and React Native (TypeScript). It lets you create and manage daily habits across three tabs: Habits, Badges, and Profile. The Habits tab shows everything scheduled for today, lets you mark habits as done or skipped, and keeps a live streak count per habit with a progress bar for the day. The Badges tab rewards consistency with trophies for streak milestones and penalizes slip-ups, so there's always something to work toward. The Profile tab gives you a shareable tag to connect with friends, lets you see their streaks, and surfaces a nudge button when someone has missed a few days so you can send encouragement.
 
## Getting Started
In order to run the application, use the makefile commands. The first command is "make install" in order to retrieve all of the necessary dependencies. Please give this command some time to load. Afterwards, you should be prepared to start the application. The second command is "make run" which will display the instructions on how to run the application through Expo Go. Please leave your terminal running during use of the app, so that connection to the server is not lost. 
 
## iOS
1. Download the Expo Go app through the App Store.
2. Using your iPhone's camera, scan the generated QR code on your terminal.
3. Allow the app to load, and look around!! Enjoy the application.
 
## Android

2. Through the Expo Go app, scan the generated QR code on your terminal.
